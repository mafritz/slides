Configurator

Warning: the configurator is in beta version and is not intended for experimental purpose.

The Configurator allows the customization of almost every aspect of the Tool in a "researcher friendly" interface. Among other things, researchers can define:

  • Affective space: it is possible to define the underlying appraisal dimensions and the number and position of the associated subjective feelings.
  • Interface: possibility to show or hide the different components of the Graphic User Interface of the tool.
  • Experimental settings: it is possible to indicate an URL for an experimental task, and also define some options such as a timer or the simulation of a fake participant's emotions.
